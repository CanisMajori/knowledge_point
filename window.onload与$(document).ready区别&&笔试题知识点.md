- window.onload与$(document).ready区别

| 对比事项 | window.onload |    $(document).ready    |
| :--: | :-----------: | :---------------------: |
| 加载时间 | 所有元素加载完毕后再去执行 | DOM结构回执完毕后(有可能图片啥的没加载好) |
|  个数  | 只会执行一个，多个会覆盖  |          可以写多个          |
|  简写  |       无       |    $(function(){代码})    |
|      |      待定       |           待定            |
